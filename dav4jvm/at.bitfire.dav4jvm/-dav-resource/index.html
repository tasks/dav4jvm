Represents a WebDAV resource at the given location and allows WebDAV requests to be performed on this resource.

Requests are executed synchronously (blocking). If no error occurs, the given callback will be called. Otherwise, an exception is thrown. These callbacks don't need to close the response.

To cancel a request, interrupt the thread. This will cause the requests to throw InterruptedException or InterruptedIOException.

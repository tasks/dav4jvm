getRange

fun getRange(accept: String, offset: Long, size: Int, headers: Headers? = null, callback: (response: Response) -> Unit)
Content copied to clipboard

Sends a GET request to the resource for a specific byte range. Make sure to check the response code: servers may return the whole resource with 200 or partials with 206.

Follows up to MAX_REDIRECTS redirects.

Parameters

accept

value of Accept header (always sent for clarity; use &#47; if you don't care)

offset

zero-based index of first byte to request

size

number of bytes to request

headers

additional headers to send with the request

callback

called with server response unless an exception is thrown

Throws

java.io.IOException

on I/O error

at.bitfire.dav4jvm.exception.HttpException

on HTTP error

at.bitfire.dav4jvm.exception.DavException

on high-level errors

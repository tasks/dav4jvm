put

fun put(body: RequestBody, ifETag: String? = null, ifScheduleTag: String? = null, ifNoneMatch: Boolean = false, callback: (Response) -> Unit)
Content copied to clipboard

Sends a PUT request to the resource. Follows up to MAX_REDIRECTS redirects.

When the server returns an ETag, it is stored in response properties.

Parameters

body

new resource body to upload

ifETag

value of If-Match header to set, or null to omit

ifScheduleTag

value of If-Schedule-Tag-Match header to set, or null to omit

ifNoneMatch

indicates whether If-None-Match: * ("don't overwrite anything existing") header shall be sent

callback

called with server response unless an exception is thrown

Throws

java.io.IOException

on I/O error

at.bitfire.dav4jvm.exception.HttpException

on HTTP error

at.bitfire.dav4jvm.exception.DavException

on HTTPS -> HTTP redirect

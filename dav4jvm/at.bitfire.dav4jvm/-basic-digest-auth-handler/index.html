BasicDigestAuthHandler

class BasicDigestAuthHandler(domain: String?, username: String, password: String, insecurePreemptive: Boolean) : Authenticator, Interceptor
Content copied to clipboard

Handler to manage authentication against a given service (may be limited to one domain). There's no domain-based cache, because the same user name and password will be used for all requests.

Authentication methods/credentials found to be working will be cached for further requests (this is why the interceptor is needed).

Usage: Set as authenticator and as network interceptor.

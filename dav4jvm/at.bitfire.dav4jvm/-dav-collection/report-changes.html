fun reportChanges(syncToken: String?, infiniteDepth: Boolean, limit: Int?, vararg properties: Property.Name, callback: DavResponseCallback): List<Property>

Sends a REPORT sync-collection request.

Return

list of properties which have been received in the Multi-Status response, but are not part of response XML elements (like sync-token which is returned as SyncToken)

Parameters

syncToken

sync-token to be sent with the request

infiniteDepth

sync-level to be sent with the request: false = "1", true = "infinite"

limit

maximum number of results (may cause truncation)

properties

WebDAV properties to be requested

callback

called for every WebDAV response XML element in the result

Throws

java.io.IOException

on I/O error

at.bitfire.dav4jvm.exception.HttpException

on HTTP error

at.bitfire.dav4jvm.exception.DavException

on WebDAV error
